Install The Concierge Widget for your users

Installing The Concierge Widget is easy. It takes just a few minutes to add the code snippet to your app or website.

Once you install The Concierge Widget, your users will be able to create a GiftStarter Campaign using any link directly from your app or website.

The basic JavaScript method is a good fit for most standard web apps which are characterized by each new screen triggering a full page refresh.

Copy and paste this code snippet before the </body> tag on every page where you want The Concierge Widget to appear.

                                        
<script type="text/javascript" id="concierge_widget_script">
    (function() {
        function async_load(){
            var s = document.createElement('script');
            s.type = 'text/javascript';
            s.async = true;
            var link = 'http://stack-builder-yaboi.c9users.io/embed/script.js';
            if (window.location.protocol == "https:") link = link.replace('http://','https://');
            s.src = link + ( link.indexOf("?") >= 0 ? "&" : "?") + 'ref=' + encodeURIComponent(window.location.href);
            var embedder = document.getElementById('concierge_widget_script');
            embedder.parentNode.insertBefore(s, embedder);
        }
        if (window.attachEvent)
            window.attachEvent('onload', async_load);
        else
            window.addEventListener('load', async_load, false);
        })();
</script>
<div id="concierge_widget_inline"></div>

To complete this step, follow the instructions above and then visit a page on your website or app where the script is installed.
